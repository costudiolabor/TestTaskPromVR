Лобби.
EntryLobby - инициализация лобби.
ScenesHandler - Загружает сцены по их индексам.
MenuLobbyHandler - Управляет UI лобби.
PlayerHandler - Создает и управляет VR-игроком.
Factory - Создание объектов (игрока, UI) через Unity-инстанцирование.
ScriptableObjects (настройки)
Prefabs – хранит префабы (например, игрока).
PrefabsUI – хранит UI-элементы (меню, панели).
SettingScenes – содержит индексы сцен.

Основная Сцена.
EntryMain - инициализация сцены.
SoundHandler - Обрабатывает вывод звуков.
Tasks - Находится на сцене, содержит настройки всех групп шагов и точек интереса. 
TaskHandler - Настраивает и обрабатывает точки интереса.
GroupHandler - Настраивает и обрабатывает группы, шаги заданий, выводит в UI группы шагов.
StatisticsHandler - Выводит в UI стасистику.
MenuGameHandler -Управляет UI.

При старте TaskHandler перебирает массив групп и шагов, присваивает им ID и событиям шагов.
Все точки интереса наследуются от базового класса StepAction. StepAction имеет событие, 
которое отправляет TaskHandler ID группы и шага. Эти данные отправляются в GroupHandler, 
где эти данные обрабатываются и выводятся в UI.
